This mini project first takes a MCQ test of n students on n subejcts(currently 3 for simplicity).

Then converts the data(marks etc) to usable form so that analysis can be done on it.

Some of the example analysis include ->

1. Girls Vs boys comparison in all subjects and overall.
2. Online course Vs non course student.
3. Toppers and Backbenchers(euphemism for failed / just passed students) overall and subject vise
4. Overall performance of class.
5. Whether being better in one subject helpsbeing good in other subject or not and so on and so forth.
